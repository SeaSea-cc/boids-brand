**Primary — homepage / brand (recommended):**
> *Build what you know. Deploy what you trust. Boids runs the loop.*

## Six-Beat Narrative Arc (v1.2)

| Beat | Content |
|---|---|
| **Pain** | Most AI resets every session. Specialized expertise doesn't travel. Developers rebuild agent infrastructure from scratch. Regular users switch between apps to access AI instead of AI coming to them. |
| **Problem** | There's no standard layer for building agents, publishing them, and getting them into the places where work actually happens — neither for the people who build products nor for the people who just want to get work done. |
| **Reveal** | Boids is the expert agent platform — a consumer product where regular users bring expert agents into Slack, Telegram, and their daily work, and a developer platform where teams build agent-powered products via API and CLI. One shared agent network underneath. |
| **Proof** | Regular users integrate expert agents into Slack and Telegram in minutes — no setup, no switching apps. Developers build on the API and distribute through the network without rebuilding the agent layer. The same agents, callable by both. |
| **Outcome** | Professional expertise becomes callable infrastructure. Developers ship faster. Users work with specialists that live where the work already happens. The agent network compounds with every new agent published. |
| **CTA** | Bring an expert agent to work — or build the next one. |

## Authoring Checklist

Before publishing any Boids copy:
- [ ] Is it clear which product line this is for? (C-end product, API / CLI, or universal / brand-level)
- [ ] Does it use the right verbs? (build / ship / distribute / call / compose / integrate / earn)
- [ ] Does it avoid consumer chat framing? (no "chat", "talk", "discover")
- [ ] Does it treat agents as callable, composable, distributable?
- [ ] Is it sentence case? No Title Case headers.
- [ ] Does it avoid "AI-powered", "cutting-edge", "revolutionary"?

---

## Forbidden Phrasings

Never use:
- "AI chatbot" / "chat with AI" / "talk to bots"
- "No-code"
- "Talk to your AI" / "discover AI agents" / "play with AI characters"
- "AI-powered" (vague — avoid)
- "Cutting-edge" / "state-of-the-art" / "revolutionary"
- "Magic"
- Any framing that reduces Boids to a single use case or consumer product
